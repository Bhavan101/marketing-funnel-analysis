# Marketing Funnel & Conversion Performance Dashboard

## 📌 Project Overview
Analysis of marketing funnel data to understand 
how users move from visitors to customers using 
Python and Power BI.

## 🛠️ Tools Used
- Python (Pandas) — Data Cleaning & Funnel Analysis
- Jupyter Notebook — Analysis
- Power BI Desktop — Dashboard

## 📊 Dataset
- Source: eCommerce Events Dataset November 2019
- Records: 1,048,575 rows
- Period: November 1, 2019
- Users: 176,639 unique customers

## 🔍 Key Insights
1. Only 1.81% of viewers make a purchase
2. 98.48% drop off without adding to cart
3. Total revenue generated = $5.65M
4. Electronics is top performing category
5. Samsung leads with 4.4K purchases
6. Peak activity hours: 2am-4am UTC

## 💡 Recommendations
1. Optimize product pages to improve cart rate
2. Add retargeting campaigns for 98% dropoff
3. Focus marketing budget on electronics
4. Create Samsung and Apple bundle deals
5. Run flash sales during peak hours
6. Simplify checkout process

## 📁 Project Structure
- funnel_summary.xlsx    → Overall funnel metrics
- category_funnel.xlsx   → Category performance
- brand_funnel.xlsx      → Brand performance
- daily_funnel.xlsx      → Hourly activity
- price_funnel.xlsx      → Price range analysis
- dashboard_screenshot.png → Dashboard preview
